# The Lay of The Land
# Required;
Create a file named system_report.txt in your current directory `(~/project).` The file must

contain the output of the `whoami`, `uname-a` (all system information), and uptime commands

Requirements:

1. The final report file must be named `system_report.txt.`
2. You must use output redirection operators `(> and >>)` to write the command outputs to the file.
3. The file must be created in the `~/project directory.`

 # Commands
 1. Creating the ~/project directory: `mkdir -p ~/project.`
 2. Creating system_report.txt and adding the output of whoami: `whoami > ~/project/system_report.txt.`
 3. Append the output of uname a:` uname -a >> ~/project/system_report.txt.`
 4. Append the output of uptime: uptime >> ~/project/system_report.txt
 5. Verify the file: `cat ~/project/system_report.txt.`
